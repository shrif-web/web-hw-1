# How run the project?


* **step #1:** Run nginx 
  * For more details check out README.md file in nginx directory.   
* **step #2:** Run both js and go back-ends.
  * In back-end-golang run `go run main.go`
  * In back-end-js run `node JsBack-end.js`
* **step #3:** check http://localhost.
